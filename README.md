# POO - Tarea Encapsulación

## 📚 Programación Orientada a Objetos

Este proyecto fue desarrollado como parte del curso de Programación Orientada a Objetos, con el objetivo de aplicar el concepto de **encapsulación** en Java mediante la creación de diferentes clases y métodos.

---

## 🎯 Objetivo

Implementar clases en Java utilizando atributos privados y métodos públicos para manipular la información, demostrando el uso correcto de la encapsulación.

---

## 🧩 Contenido del Proyecto

El proyecto incluye las siguientes clases:

### 👤 Persona
- Atributos: nombre, apellido, edad
- Método: mostrar()
- Permite visualizar la información de una persona

### 📐 Rectángulo
- Atributos: base, altura
- Métodos: area(), perimetro()
- Permite calcular el área y perímetro

### 🛒 Producto
- Atributos: código, nombre, precio
- Método: aplicarDescuento()
- Permite aplicar descuentos al precio

### 💰 Cuenta
- Atributos: titular, saldo
- Métodos: depositar(), retirar()
- Controla operaciones bancarias sin permitir saldo negativo

### 📚 Libro
- Atributos: título, autor, páginas
- Método: esLargo()
- Determina si un libro tiene más de 300 páginas

### 🧮 Calculadora
- Métodos: sumar, restar, multiplicar, dividir
- Incluye validación de división entre cero

### 📅 Fecha
- Atributos: día, mes, año
- Método: esValida()
- Verifica si una fecha es válida según rangos básicos

---

## ▶️ Ejecución

El programa se ejecuta desde la clase `Main.java`, donde se crean objetos de cada clase y se prueban sus métodos, mostrando resultados en consola de forma ordenada.
